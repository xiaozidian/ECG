# real_data - 医院真实患者心电数据

## 数据来源
医院提供的患者级心电数据，用于模型验证和报告生成。

## 目录结构
共 20 个患者文件夹，按患者 ID 命名：
- `2508040816266615/` ~ `2508040930572143/`（第一批 10 例）
- `2602020832425821/` ~ `2602020930098424/`（第二批 10 例）

每个患者文件夹（约 290-320 MB）包含：
- `.DATA` - 原始 ECG 二进制数据（int16 格式，多导联）
- `.EXAM` - 检查信息文件
- `.PAC` - 打包数据
- `*_beat_annotations.csv` - 心搏标注
- `*.png` - 可视化图片
- `DGS/` - 诊断子目录
- `report/` - 报告子目录

## 总大小
约 5.9 GB

## 如何获取
此数据为医院临床数据，未上传至 GitHub。请联系数据提供方获取。
另有额外 10 例数据压缩包：`另10个数据20260203.rar`（同级目录，约 814 MB）

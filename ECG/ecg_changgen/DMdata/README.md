# DMdata - 动态心电（Holter/DM）原始数据

## 数据来源
医院提供的动态心电记录仪原始数据。

## 目录结构
- `Save000/` ~ `Save032/`：33 个患者的 Holter 记录，每个文件夹约 170-185 MB
- `group2/`：第二组患者数据（约 6.6 GB）

## 文件格式
每个患者文件夹包含：
- `.dat` - 原始 ECG 波形数据（二进制）
- `.dbf` - 数据库文件（患者信息/标注）
- `.def` - 设备定义文件

## 总大小
约 13 GB（压缩包 DMdata.zip 约 5.9 GB）

## 如何获取
此数据为医院临床数据，未上传至 GitHub。请联系数据提供方获取，或从服务器备份恢复。
原始压缩包：`DMdata.zip`（同级目录，约 5.9 GB）
